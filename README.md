# password-generator
Save and create passwords using Tkinter interface.
Allows user to save website and password combinations to their computer using a GUI. Also allows retrieval of pre-existing passwords and generation of randomised passwords. New passwords automatically copied to Clipboard using pyperclip package.

Packages used: Tkinter, pandas, random, pyperclip, json
Skills used: random package, Tkinter interface (pop-up message boxes, buttons, user inputs), exception handling, JSON files (read, write, append)
